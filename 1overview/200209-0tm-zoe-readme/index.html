TM 时间账单服务¶

背景¶

  • 最早的时帐任务:
    • 28d[TASK]时间账单任务 (原 S01E01 任务 -> S01E01.2)
    • [ASK] 有靠谱的时间记录工具否?
  • 原本是想先通过手工时帐体验, 引发工具探索
  • 进而形成共同的工具以及规约
  • 没想到太过艰难, 无人坚持

目的¶

通过100天时帐记录和3次分析反馈

  • 建立私人时帐习惯
  • 熟悉怼圈 trao 工具
  • 完成时帐记录->反馈->持续优化 循环
  • 形成通过时间帐单持续优化私人效能案例
    • 为明年发布自怼时间管理课程作准备
